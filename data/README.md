# Data Folder

Download the dataset from Kaggle and place the CSV file in this folder before running the workflow:

https://www.kaggle.com/datasets/rauffauzanrambe/fifa-world-cup-2026-player-performance-dataset

Expected filename:

`fifa_world_cup_2026_player_performance-selected-columns-2.csv`

The CSV is intentionally not committed to this repository because the dataset license should be checked on Kaggle before redistributing the data publicly.
